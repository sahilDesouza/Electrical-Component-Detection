# Electrical-Component-Detection
ML model developed to detected and differentiate between an led, diode and background
The model is initially trained using these 3 data sets but can be updated to detected many more electrical components
Data acquisition was done using openmv H7 Plus and script to create a snapshot of desirable number is added to this repositiory
Additionally the .tflite model is added as well for users to test. This model was created using edge impulse where the neural network was trained
A script to detect the output based on these 3 components was also added
